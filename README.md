# DG1IAD_Portfolio3
AstonCV, a CV databased website 

## Requirements for Public
- View a list of all CVs showing basic information (name, email, key programming language)
- CV view details (basic information + education, profile, links, etc.)
- Search CVs using name / key programming language 
- Register to become a registered user

## Requirements for Registered Users
- Log in system
- Updating CV by inputting more details
- Logout system

## Security Measures
- Authentication
- Authorization
- Form validation
- Handle injections (SQL/HTML)
- Hash password
- CSRF
